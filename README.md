# 📚 DSA in C++

## 🚀 Goal

This repository is dedicated to learning **Data Structures and Algorithms (DSA)** by:

* implementing each data structure from scratch in C++
* solving problems to understand real-world usage
* building strong fundamentals for interviews and problem solving

## 🧠 Learning Approach

For each topic:

1. Understand the concept
2. Implement from scratch in C++
3. Compare with STL
4. Solve related problems
